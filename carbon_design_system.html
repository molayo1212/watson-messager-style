사용 가이드

이 스타일 시트 사용하기

1. 기본 사용법

이 스타일 시트를 HTML 파일에 포함하거나 CSS를 복사해서 사용하세요.

2. 다크 테마 적용

<body class="watson-dark-theme">를 추가하면 다크 테마가 적용됩니다.

3. 주요 클래스

  • .cds-text-input - Carbon 스타일 텍스트 입력
  • .cds-btn--primary - 기본 버튼
  • .cds-inline-notification--error - 에러 알림
  • .watson-logo - Watson 로고

4. 재사용 가능한 변수

모든 색상, 간격, 타이포그래피가 CSS Custom Properties로 정의되어 있어 쉽게 커스터마이징할 수 있습니다.
